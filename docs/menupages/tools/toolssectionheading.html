Tools Menu Options

The Tools Menu contains the tools available for use in and with PCGen.

The Treasure Generator is used to randomly generate treasure to be added to any character.

The Filters menu options are for use within PCGen to add, edit, or create custom filters for sorting items on the various character tabs.

GMGen opens GMGen, the game master aid.

Install Dataset allows the user to install datasets released out of the normal cycle of release for PCGen.

The List Editors menu option is populated by a list of editors for users to enter/edit the list files.
